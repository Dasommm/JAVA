Program : 실행할 수 있는 파일(.exe)
Process : 응용프로그램(.exe) 단위로 실행되는 모듈 (=실행되고 있는 상태)
Thread : 하나의 Process에 여러개의 메서드 단위로 실행되는 모듈 ( 하나의 작업단위)

Process - T,T,T,...(os에서 제어할 수 있는 가장 작은 단위)

---------------------------------------------------------
자바에서 메서드 단위로 프로그램을 실행하는 방법 2가지
1. Runnable 를 implements : 단순한 쓰레드화
2. Thread를 extends : 쓰레드의 기능 사용
*run() -> Thread.start() 메서드가 자동 호출해준다.
